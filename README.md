# thameem.me

Built with [Next.js](https://nextjs.org/) and [Tailwind CSS](https://tailwindcss.com/), deployed
on [Vercel](https://vercel.com/).

## Pages

| Route       | Description                                       |
|-------------|---------------------------------------------------|
| `/`         | Home — intro, experience, and about snapshot      |
| `/about`    | Full bio, skills, conferences, and community work |
| `/contact`  | Contact form                                      |
| `/resume`   | Resume                                            |

## Tech Stack

- **Framework** — Next.js 13
- **Styling** — Tailwind CSS
- **Language** — TypeScript
- **Deployment** — Vercel

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view locally.

## Scripts

```bash
npm run dev      # Start development server
npm run build    # Build for production
npm run start    # Start production server
npm run lint     # Run ESLint
```
